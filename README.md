## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Run the cloudlfare simulated version locally to debug issues before they break your production 😔 :

```bash
npm run preview # or pnpm, yarn, bun preview
```

This project aims to collect the latest open source contribution opportunities across Github and Gitlab. You can bookmark your favorite issues to visit them later. You can search for similar issues by cliking on available labels.
Find the live website at: [https://goodfirstissues.net](https://goodfirstissues.net/)
